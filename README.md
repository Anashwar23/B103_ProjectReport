**E-Commerce Database Management System**

Overview:
This project aims to implement a relation database system for an e-commerce platform.
It tries to manage customers, products, orders, order details and transactions whil ensuring data integrity through constraints and references 


Files:
1. E-commcerceSchema: Contains the structure of the database management system
   
2. Data_Insertion: Contains query on inserting information in the table
   
3. Basic_Data_Retrieval.sql : Contains query that displays inserted data

4. Conditional_filtering_data_retrieval: Contains query to display filtered data

5. Aggregation_Queries: Contains query that display operational calculations on data

6. Left_Join: Contains query to join left matching table with another

7. Right_Join: Contains query to join right matching table with another

8. InnerJoin: Contains query to display an Inner Join of 2 tables

9. Case_Query: Contains a query displaying the case functionality

10. If_Query: Contains a query with a logical block

11. Procedure_Query: Contains an executable procedure block

12. Trigger_Query: Contains a trigger that runs automatically

13. Transactions_Query: Contains a control transaction query that would rollback if failed.
